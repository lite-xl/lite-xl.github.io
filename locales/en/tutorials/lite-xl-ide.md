# Setup Lite XL as an IDE

Lite XL can be configured to be a valid (WIP) alternative to the most popular IDEs like VSCode, Eclipse Java Edition, the Netbrains suite, etc.

The following features can be enabled through a combination of installed plugins and configuration instructions inside Lite XL's `init.lua`:

- Syntax highlighting
- Smart autocompletion
- Error highlighting
- Code snippets
- Integrated terminal
- VSCode-like minimap
- Git integration

And much more.

There is an attempt at setting up a comprehensive and straightforward setup [here][1].

There is also an attempt at setting up a more focused (C/C++) setup [here][2], with custom plugins for building and debugging.
